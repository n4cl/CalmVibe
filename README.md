# CalmVibe

心拍ガイド（BPMベース）と呼吸ガイド（吸・止・吐フェーズ）で落ち着きを取り戻すためのシンプルなモバイルアプリ。
現在は MVP 開発中です。

## 状態
- プラットフォーム: Expo（React Native / Web対応）
- コア機能: 心拍ガイドと呼吸ガイドの開始・停止、設定保存
- 進捗: 履歴まわりは実装中（タスク 5.x）
